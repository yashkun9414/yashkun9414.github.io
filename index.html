Project Evolution

Key Deviations
Dropped some advanced analytics features due to time constraints, while focusing more on reliability and real-time alert delivery. This strategic pivot ensured a robust, production-ready system optimized for emergency response.
Sustainability Approach
Cloud-based architecture ensures scalability and easy extension to cover additional states. System designed for seamless integration with government disaster management infrastructure, ensuring long-term adoption and widespread impact.

Key Learnings

Technical Achievements
  • Mastered cloud deployment and horizontal scalability patterns

Strategic Insights
